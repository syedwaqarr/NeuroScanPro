Overview

This project focused on developing a robust image classification model to differentiate between 'Healthy' and 'Tumor' images using a dataset of over 4500+ samples. The model achieved exceptional performance with an overall accuracy of 98%. The healthy and tumor categories were effectively distinguished, demonstrating the model's reliability in medical image classification tasks.

Tools & Technologies

Numpy, pandas, seaborn, Matplotlib, scikit-learn, PyTorch, Pillow, JavaScript CSS, HTML, Google Colab, Kaggle
